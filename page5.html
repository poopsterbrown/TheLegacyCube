AfterThought (formally known as The Legacy Cube) minimum requirements for Windows:
Minimum System Requirements:

OS: Windows 7/8/8.1/10/11 (32bit/64bit)
Processor: Intel Core 2 Duo or better
Memory: 4GB RAM
Graphics: DirectX 9/OpenGL 4.1 capable GPU
Storage: Approximately 100GB pre-installation. 50GB after if you delete the install zip file.
